# routertl-ip-index

A curated, version-locked index of FPGA IP packages for use with the [RouteRTL SDK](https://github.com/tich-tech/routertl).

## What Is This?

`routertl-ip-index` is a **data-only registry** — it does not re-host IP source code. Each package is a YAML descriptor that declares:

- Where to fetch the IP (git URL + pinned commit SHA)
- Which HDL files to include
- Compatibility metadata for the RouteRTL build system (`library`, `language`)

The `catalog.json` at the root is the machine-readable index, auto-regenerated by CI whenever any package YAML changes.

## Package Schema

Each version file lives at `packages/<namespace>/<name>/<version>.yml`:

```yaml
name: olo_axi_lite_slave
namespace: open-logic
version: "2.5.0"
description: "AXI4-Lite slave port — register-map compatible, configurable address/data width"
license: LGPL-2.1-or-later

source:
  type: git
  url: https://github.com/open-logic/open-logic.git
  tag: v2.5.0          # semver tag — use 'commit:' for repos with no tags
  ip_yml_path: "."

routertl_compat:
  library: olo          # VHDL/Verilog library name for compilation
  language: vhdl        # vhdl | verilog | systemverilog | mixed
  files:
    - src/base/vhdl/olo_base_pkg.vhd
    - src/axi/vhdl/olo_axi_lite_slave.vhd
```

### Source Pinning Rules

| Repo type | Field to use | Why |
|:----------|:-------------|:----|
| Semver tags available | `tag: v2.5.0` | Stable, human-readable |
| No semver tags (e.g. ZipCPU) | `commit: <full SHA>` | Reproducibility — `tag: master` is **forbidden** |

Update a pinned commit:
```bash
git ls-remote https://github.com/<org>/<repo>.git HEAD
# Copy the SHA into the package yml
python3 generate_catalog.py
```

## Adding a New Package

1. Create `packages/<namespace>/<name>/<version>.yml`
2. Follow the schema above — fill in `source`, `routertl_compat.language`, and `files`
3. Create `sources/<namespace>.yml` if the namespace is new
4. Run `python3 generate_catalog.py` locally to verify
5. Commit — CI will regenerate and commit `catalog.json` automatically

## Regenerating Locally

```bash
pip install pyyaml
python3 generate_catalog.py
```
